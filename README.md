## Getting Started

### Installation

1. Clone the repo
2. install dependencies
   ```sh
   npm install
   ```
3. create an .env file in the root directory filling in your personal DB_host (MongoDB):

   ```
   # Database
   DB_HOST=your.db.here

   ```
